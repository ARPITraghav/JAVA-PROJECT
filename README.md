# JAVA-PROJECT
A console-based Online Quiz System developed in Java. The program presents multiple-choice questions, evaluates user answers, calculates the score, includes a countdown timer using multithreading, and stores results using file I/O.

## Setup Instructions

1. Install **Java JDK** on your system.
2. Clone this repository
```
git clone https://github.com/your-username/Online-Quiz-System-Java.git
cd Online-Quiz-System-Java
```
3. Compile the program
```
javac Main.java
```
4. Run the program
```
java Main
```
5. Enter your name and start the quiz.
